ALGORITHM TEST

  1. Connect to the local server as instructed for necessary information of the competition.
  2. Read the instructions carefully.
  3. Find the link for programming challenges.
  4. Write the programs in your system.
  5. Connect to the local server and transfer the folder Competition to your local machine.
  6. Test the correctness of your program using a script testYourCode.exe with the help of ReadMe. txt file present in Competition folder.
  7. After successful testing copy source file (.c or .cpp) and executable file (.exe) to the Result folder.
  8. Transfer your Result folder to local server.
